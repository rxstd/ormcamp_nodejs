#### ECMAScript 2015 (ES6) 주요 문법.
 ## 신규 변수, 상수 사용법.
  - var => 재 선언과 재 할당이 전부 가능한 "변수" 선언법이다.
  - let => 재 선언이 불가하고 재 할당만 가능한 "변수" 선언법이다.
  - const => 재 선언과 재 할당이 전부 불가능한 "상수" 선언법이다.


#### 기타 여러 내용.
 ## HTML 내 소소한 내용들.
  - html 파일 최상단에 <!DOCTYPE html> 이라고 적는것은 html5임을 의미한다.
    <!DOCTYPE html5> 가 아니라 그냥 <!DOCTYPE html>로만 적는 이유는 html5가 최초로 규격화된 버전이기 때문이다.

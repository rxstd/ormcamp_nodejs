#### ECMAScript 2015 (ES6) 주요 문법.
 ## 신규 변수, 상수 사용법.
  - var => 재 선언과 재 할당이 전부 가능한 "변수" 선언법이다.
  - let => 재 선언이 불가하고 재 할당만 가능한 "변수" 선언법이다.
  - const => 재 선언과 재 할당이 전부 불가능한 "상수" 선언법이다.
  - var는 함수 레벨 스코프를 가지며, 호이스팅이 발생하여 변수 선언이 해당 스코프의 최상단으로 올라온다.
  - let과 const는 블록 레벨 스코프 개념이 적용된다. 호이스팅이 발생하긴 하지만 초기화 선언 전에는 접근할 수 없다.
  - const는 객체를 선언할 때도 사용된다. 재할당과 재선언은 불가능하지만 속성은 수정할 수 있다.

#### 클라이언트 (Client) 와 서버 (Server) 의 개념 이해. client_server.png 참조
 ## 기본적인 소통 시나리오
  1. 클라이언트 측에서 여러 방법을 통해 서버에 요청을 한다. 이는 여러 운영체제 혹은 소프트웨어를 거치게 된다.
  2. 서버의 엔드포인트에 원하는 정보를 페이로드를 담아 요청하게되면 서버에서 여러 포맷을 통해 응답하게된다.
  3. 서버에서 정보를 리턴하면 클라이언트에서는 해당 정보 객체를 가지고 정보를 재배열하고 배치하여 사용자에게 제공하게된다.
